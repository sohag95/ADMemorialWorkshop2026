# Ajit Diwan Memorial Workshop 2026

This repository contains the complete source files for the official webpage of the Ajit Diwan Memorial Workshop 2026.

The website presents essential information related to the workshop, including the schedule, speakers, and general event details, in a clean and accessible format.

## Repository Structure

.
├── index.html      # Main HTML file for the workshop webpage 

 ├── styles.css      # CSS file for styling and layout 

 ├── images/         # Images used on the website  

 └── README.md       # Project documentation

## Website Content

The webpage includes:
- Workshop title and overview
- Schedule / timetable of sessions
- Speaker information
- Committee details

## Technologies Used

- HTML5
- CSS3

## How to View the Website

Open the file `index.html` in any modern web browser.

(Optional) Run a local server:
python -m http.server 8080
Then visit:
http://localhost:8080

## Notes

- This is a static website.
- All images should be placed inside the `images` folder.
- Updates can be made by editing the HTML and CSS files.
